# Rail Drishti AI - Project Summary

## Executive Overview

**Rail Drishti AI** is an AI-powered railway assistance platform that combines Machine Learning and Conversational AI to improve railway-related user experience.

The project provides two major functionalities:

1. **Train Delay Prediction System**
   Predicts expected train delays using historical railway data and machine learning models.

2. **AI Railway Chatbot**
   An intelligent chatbot built using LangChain and Gemini Flash 2.5 for answering railway-related queries.

The platform is designed with a modern frontend, Python backend, and modular architecture suitable for learning, demonstration, and future scalability.
